# Security Policy

## Supported Versions

| Version   | Supported          |
| -------   | ------------------ |
| 1.2.x     | :white_check_mark: |
| 1.1.x     | :white_check_mark: |
| < 1.1.0   | :x:                |

## Reporting a Vulnerability

You can easily create an issue with vulnerability label to
report any security issue in any package.
